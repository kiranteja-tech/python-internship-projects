# Excel Report Generator using Python

## Project Overview

The Excel Report Generator is a desktop-based data analysis application developed using Python. The project automates the process of transforming raw CSV data into a structured and professionally formatted Excel report. It provides statistical summaries, pivot table analysis, and graphical visualizations through an easy-to-use graphical interface.

The application is designed to reduce manual data processing effort and demonstrate practical usage of Python libraries for real-world data engineering and reporting tasks.

---

## Objective

The main objective of this project is to:

* Read real-world dataset files in CSV format
* Perform automated data analysis
* Generate statistical summaries
* Create pivot tables for aggregated insights
* Visualize data using charts
* Export results into a formatted Excel report
* Provide a user-friendly GUI for interaction

---

## Features

* Graphical User Interface built using Tkinter
* CSV file selection through file dialog
* Automatic detection of numeric and categorical columns
* Statistical calculations including:

  * Total

* Pivot table generation for summarized analysis

import numpy as np 
import matplotlib.pyplot as plt 

class NeuralNetwork: 

    def __init__(self, input_size, hidden_size, output_size, 
                 activation='tanh'): 
        
        # Initialize weights and biases 
        self.weights_input_hidden = np.random.randn(input_size, 
                                                    hidden_size) 
        
        self.bias_hidden = np.zeros((1, hidden_size))

        self.weights_hidden_output = np.random.randn(hidden_size, 
                                                     output_size) 
        
        self.bias_output = np.zeros((1, output_size)) 
        
        self.activation = activation 

    def tanh(self, x): 
        return np.tanh(x) 

    def forward(self, X): 
        
        # Forward pass 
        self.hidden_input = np.dot(X, self.weights_input_hidden) + \
                            self.bias_hidden 
        
        if self.activation == 'tanh': 
            self.hidden_output = self.tanh(self.hidden_input) 
        
        else: 
            raise ValueError("Invalid activation function") 
        
        self.output = np.dot(self.hidden_output, 
                             self.weights_hidden_output) + \
                             self.bias_output 
        
        return self.output 


# Example usage: 

input_size = 3 
hidden_size = 4 
output_size = 2 

# Create a neural network with tanH activation function 
nn_tanh = NeuralNetwork(input_size, hidden_size, output_size, 
                        activation='tanh') 

# Sample input data 
X = np.array([[0.1, 0.2, 0.3]]) 

# Forward pass with tanH activation 
output_tanh = nn_tanh.forward(X) 

print("Output with tanH activation:", output_tanh)


# Plotting tanH Activation Function 

# Generate x values 
x_values = np.linspace(-10, 10, 100) 

# Calculate y values for tanH activation function 
y_tanh = np.tanh(x_values) 

# Plot the tanH activation function 
plt.figure(figsize=(6, 4)) 

plt.plot(x_values, y_tanh, label='tanH', color='purple') 

plt.title('tanH Activation Function') 
plt.xlabel('Input') 
plt.ylabel('Output') 

plt.grid(True) 
plt.legend() 

plt.tight_layout() 
plt.show()
